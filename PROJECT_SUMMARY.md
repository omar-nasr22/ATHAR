## 🏗️ البنية التقنية

### الواجهة الأمامية (Frontend)
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **HTTP Client:** Axios
- **Internationalization:** i18next
- **Routing:** React Router
- **Icons:** Lucide React

### السيرفر الخلفي (Backend)
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT
- **File Uploads:** Multer
- **Email:** Nodemailer
- **Payment:** Stripe

## 🚀 كيفية التشغيل

### 1. تشغيل السيرفر الخلفي
```bash
cd server
node index.js
```
السيرفر سيعمل على: `http://localhost:5000`

### 2. تشغيل الواجهة الأمامية
```bash
cd client
npm run dev
```
الواجهة الأمامية ستعمل على: `http://localhost:5173`

### 3. تسجيل الدخول كأدمن
- افتح: `http://localhost:5173/admin/login`
- البريد: `admin@athar.ai`
- كلمة المرور: `admin123`
